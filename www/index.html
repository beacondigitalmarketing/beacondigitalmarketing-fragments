fragments, the front-end component library for 24 ways

You are viewing the front-end component library for 24 ways. It is powered by Fractal, a tool that enables the rapid development of components, templates and pages.

Component categories

  • Utilities: Components with a single purpose and specific role
  • Common: Components that may appear anywhere on the site
  • Global: Components that appear on every page on the site
  • Scopes: Components used to style scoped areas of content
  • Templates: Templates for particular page types

The source code is available on GitHub. See the README for installation instructions.
